Penjelasan GF(2⁴)

GF(2⁴) adalah Galois Field dengan 16 elemen, dipakai sebagai basis aritmatika seluruh operasi nibble pada S-AES (SubNibbles menggunakan pemetaan tabel, sedangkan MixColumns menggunakan perkalian field secara langsung).

  • Penjumlahan (GFAdd): setara operasi XOR bit-per-bit.
  • Perkalian (GFMul): perkalian polinomial biasa, direduksi modulo polinomial irreducible x⁴+x+1 (0x13) apabila derajat hasil ≥ 4.
  • Elemen identitas perkalian: 0001 (nilai 1).
  • Elemen nol: 0000.
Tata Letak State Matrix

16-bit input (plaintext / ciphertext / key) dipecah menjadi 4 nibble n0 n1 n2 n3 dari bit paling signifikan ke paling tidak signifikan, kemudian disusun ke dalam matriks state 2×2 secara kolom-mayor:

Input 16-bit
